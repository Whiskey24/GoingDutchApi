Read Before you Update - New Users ignore

Inside the uFlex Class file there is constant declared named salt (uFlex::salt). It is strongly adviseble that you change its value as it will make your authentication systen unique. If you did changed it please follow this steps to upgrade:

  • Rename your current class.uFlex.php file to class.uFlex.old.php
  • Copy he new class.uFlex.php file to the desire directory
  • Copy the custom value of the salt constant in class.uFlex.old.php
  • Paste it in the new file class.uFlex.php
That Should be it.
Failing to do this will render your existing database useless.
